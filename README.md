# FNP Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes sales performance of Ferns N Petals (FNP) using Microsoft Excel.

The dashboard provides insights into revenue, order trends, delivery performance, occasions, product categories, cities, and monthly sales patterns.

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Timeline Filters
- Dashboard Design

## 🔍 Key Insights

- Anniversary generated the highest revenue.
- Colors category achieved the highest sales.
- August was the highest revenue month.
- Tuesday recorded the highest revenue among weekdays.
- Imphal and Dhanbad were among the top-performing cities.
